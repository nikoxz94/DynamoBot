# DynamoBot

DynamoBot is a Twitch moderation and utility bot designed to automate chat management and community interaction.

## Core Features
* **Automod:** Automatic filtering of banned words and phrases.
* **Anti-Spoiler:** Specialized filter to detect and block spoilers in chat.
* **Commands:** Includes built-in commands like `!so` (shoutout) and `!social`.

## Current Status
* **UI/UX:** There is currently **no user interface** implemented. The bot is managed and configured directly via code/configuration files.
* **Development:** Functional core for Twitch API integration.

## Tech Stack
* Typescript
* Twurple libraries
